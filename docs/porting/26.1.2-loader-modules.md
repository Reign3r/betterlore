# Better Lore 26.1.2 loader modules

This patch turns the former single Fabric Gradle project into the first
MultiLoader-style 26.1.2 workspace.

## New module shape

```text
fabric/
  Builds Fabric 26.1.2 from src/common + src/fabric.

neoforge/
  Builds NeoForge 26.1.2 from src/common + src/neoforge.

forge/
  Builds Forge 26.1.2 from src/common + src/forge.

src/common/
  Shared anvil logic, UI, QuickText-compatible parser, item storage,
  component writing, JEI exclusion plugin, mixins, assets, and translations.

src/fabric/
  Fabric entrypoints, Fabric networking, Fabric QuickText adapter,
  Fabric REI plugin, Fabric service providers, and fabric.mod.json.

src/neoforge/
  NeoForge entrypoint, NeoForge networking, NeoForge service providers,
  and neoforge.mods.toml.

src/forge/
  Forge entrypoint, Forge networking, Forge service providers, and mods.toml.
```

## Build tasks

Once the Gradle wrapper jar is restored, these are the intended local checks:

```bash
./gradlew :fabric:build
./gradlew :neoforge:build
./gradlew :forge:build
./gradlew buildAll26_1_2
./gradlew testAll26_1_2
```

## Current state

The module layout is now present, but the Forge and NeoForge modules should be
considered draft modules until a real Gradle build is run against their userdev
APIs.

The common logic was intentionally kept unchanged. The loader modules only wire:

- entrypoints
- networking
- service providers
- metadata
- compile/runtime dependencies

## QuickText policy

Fabric continues to use Text Placeholder API when present. Forge and NeoForge
currently use Better Lore's clean-room QuickText-compatible fallback parser. The
fallback is implemented from public behavior and color math; it is not copied
from Placeholder API source.

## Recipe viewer policy

JEI exclusion support is shared through the common JEI plugin. Fabric REI
support remains in the Fabric source set. NeoForge REI is marked in the version
matrix and should be wired after the first NeoForge compile passes. Forge REI
will be added only after a tested Forge REI API artifact is selected for
Minecraft 26.1.2.
